Although Rails is a fairly new framework, we at railsdoctors already have several years of experience with developing rails applications in enterprise environments. In the process we have developed several tools, like the widely used request log analyzer gem.

If you are currently running a rails application in the wild and have either performance or security concerns, you have come to the right place.

Railsdoctors can analyze your application and offer advice on caching, database design, application structure and even server architecture. If you want additional information, please contact us through the contact page.
