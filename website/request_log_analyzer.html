Request log analyzer is an opensource tool the railsdocotors created to analyze applications. It is a simple command line tool which can be used to analyze request log files of both Rails and Merb to produce a performance report. Its purpose is to find what actions are best candidates for optimization.

You can install it by running sudo gem install request-log-analyzer or fetch the source from Github.
